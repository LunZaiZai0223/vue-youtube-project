# [Vue] vue-youtube-project

## About

- 練習串接 Youtube API，抓取臺灣熱門前 200 部影片
- 練習 Vue 的使用
- 初次使用 [Intersection Observer API](https://developer.mozilla.org/en-US/docs/Web/API/Intersection_Observer_API)，實作無限下拉取得 API 資料 + 圖片 lazy loading
